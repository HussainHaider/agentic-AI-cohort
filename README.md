# Agentic AI Cohort — Learning Repository

This repository is a hands-on learning workspace for building **agentic AI systems** step by step, from prompt basics to multi-agent workflows using different frameworks.

---

## What this repository is for

This is a **learning repo** focused on:

- understanding LLM-powered assistants
- learning tool/function calling patterns
- building practical AI assistants for real tasks
- experimenting with orchestration frameworks like **LangGraph** and **CrewAI**

It is organized by sessions so learners can progress from fundamentals to advanced agent architecture.
